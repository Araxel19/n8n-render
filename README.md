# n8n en Render (Deploy Fácil)

Este proyecto contiene los archivos mínimos para desplegar n8n en Render usando la imagen oficial de Docker.

## Pasos
1. Crear un repositorio con estos archivos.
2. Conectar Render → New Web Service → Deploy from GitHub.
3. Render leerá `render.yaml` y desplegará el servicio.
4. Acceder a n8n: https://<tu-dominio>.onrender.com
